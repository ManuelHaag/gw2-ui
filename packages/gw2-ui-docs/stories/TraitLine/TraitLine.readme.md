## Usage

```js
import { TraitLine } from 'gw2-ui';
```

## Properties

| Property   | Type              | Default | Required | Description            |
| ---------- | ----------------- | ------- | -------- | ---------------------- |
| `id`       | `number`          | -       | yes      | The specialization id  |
| `selected` | `array of number` | -       | -        | The selected trait ids |
