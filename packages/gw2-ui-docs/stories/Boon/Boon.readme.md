## Usage

```js
import { Boon } from 'gw2-ui'; // or gw2-ui-components
```

## Properties

| Property | Type     | Default | Required | Description   |
| -------- | -------- | ------- | -------- | ------------- |
| `name`   | `string` | -       | yes      | The boon name |

The remaining props will be spread to the underlying `<Effect/>` component.
